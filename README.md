## 📚 Sobre o Projeto

Este projeto é uma aplicação web desenvolvida em **Java** utilizando tecnologias da plataforma **Jakarta EE**, como **JSP e Servlets**.  
A aplicação é executada em um servidor **GlassFish** e foi desenvolvida utilizando a IDE **Apache NetBeans**.

O projeto está sendo desenvolvido como parte de um **trabalho acadêmico**, com o objetivo de aprender e praticar conceitos de desenvolvimento web com Java.

## ⚙️ Como Executar

### Pré-requisitos

Certifique-se de ter instalado:

- Java JDK 17 ou superior
- Apache NetBeans
- GlassFish Server

### Passos para execução

1. Abra o **Apache NetBeans**
2. Clique em **File → Open Project**
3. Selecione a pasta do projeto
4. Clique com o botão direito no projeto
5. Clique em **Run**

O servidor **GlassFish** será iniciado automaticamente.
